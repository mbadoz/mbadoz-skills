# mbadoz-skills

A curated collection of Claude Code skills.

## Skills included

| Skill | Description |
|-------|-------------|
| `3d-pixel-web` | Create stunning 3D websites with pixel art and mosaic UI/UX using Three.js |
| `adaptive-learner` | Meta-skill for continuous AI self-improvement through structured feedback |
| `built-planner` | Comprehensive task planning with detailed execution plans |
| `cron-openclaw` | Create and manage cron jobs and scheduled tasks in OpenClaw |
| `openclaw-skelette` | Expert in creating/auditing OpenClaw agent skeletons |
| `pixel-art-ui-portfolio` | Design narrative pixel art portfolio websites using PixelLab API |

## Installation

First, add the marketplace:

```bash
/plugin marketplace add mbadoz/mbadoz-skills
```

Then install only the skills you want:

```bash
/plugin install 3d-pixel-web@mbadoz-skills
/plugin install adaptive-learner@mbadoz-skills
/plugin install built-planner@mbadoz-skills
/plugin install cron-openclaw@mbadoz-skills
/plugin install openclaw-skelette@mbadoz-skills
/plugin install pixel-art-ui-portfolio@mbadoz-skills
```

## Usage

After installation, run `/reload-plugins` to activate. Skills trigger automatically based on your task context.
